# UID-PROJECT
# YONEX Website Clone 🏸

> A front-end clone of the official [Yonex](https:////github.com/vamsi-yerra/UID-PROJECT) sports brand website, built with pure HTML, CSS, and JavaScript.

---

## 📁 Project Structure

```
yonex-clone/
├── index.html       # Main HTML structure
├── style.css        # All styling and layout
├── script.js        # Interactivity and scroll logic
└── README.md        # Project documentation
```

---

## ✨ Features

- **Sticky Navigation Bar** — Fixed header with logo, nav links, and icon actions
- **Scroll Progress Bar** — Green progress indicator at the top as you scroll
- **Hero Section** — Full-viewport banner with headline, badge, and CTA button
- **Two-Column Featured Section** — Tennis & Snowboarding product cards with hover zoom
- **MUSE Section** — Highlighted product launch section with dual CTA buttons
- **Badminton Essentials Grid** — 5-product responsive grid with images and Shop Now links
- **Athletes Section** — Showcase of Yonex-sponsored athletes with photos
- **Footer** — Multi-column footer with links, newsletter input, and copyright
- **Search Functionality** — Click the search icon to query Yonex's official site
- **Smooth Scroll** — Anchor links scroll smoothly to sections

| Technology | Usage |

## 🚀 Getting Started

1. **Clone or download** the repository
2. Open `index.html` directly in any modern browser
3. No build tools, no dependencies, no install required

```bash
# Optional: serve locally with VS Code Live Server or Python
python -m http.server 8000
```

---

## 🎨 Design Highlights

- Brand color `#00A651` (Yonex Green) used consistently across buttons, progress bar, and accents
- Responsive product grid using `auto-fit` and `minmax`
- Card hover effects with `transform: translateY` and `scale`
- Dark overlay gradient on product cards for text legibility
- Mobile-friendly meta viewport tag included
